# 基于 DeepSeek 的智能代码审查多 Agent 系统

## 项目介绍
这是一个使用 DeepSeek 大模型驱动的**多 Agent 代码审查系统**，能够自动分析代码、检查规范问题、识别潜在风险，并生成优化建议。

## 核心功能
- 自动进行代码规范检查
- 支持多 Agent 协作（静态分析 + 架构评估 + PR 生成）
- 自动重试与错误恢复机制
- 支持长链推理和闭环处理

## 技术栈
- **核心模型**：DeepSeek Chat
- **语言**：Python
- **架构**：多 Agent 协作 + ThreadPoolExecutor 并行处理

## 使用场景
主要用于团队代码 Review 提效，帮助开发者快速发现代码问题，提升代码质量和 PR 通过率。
